HTML Entities

HTML entities are codes used to represent special characters and symbols that have reserved meanings in HTML.

HTML entities are represented using an ampersand (&) followed by a code and a semicolon (;).


< :- Less than sign(<)
> :- Greater than sign (>)
& :- Ampersand (&)
" :- Double quotation mark (")
' :- Single quotation mark or apostrophe (')
  :- Non-breaking space ( )
$ :- Dollar Sign ($) (Using Named Entity)
$ :- also Dollar sign ($) (Using Hexadecimal Numeric Reference)
$ :- also Dollar sign ($) (Decimal Numeric Reference)
© :- Copyright symbol (©)
® :- Registered trademark symbol (®)
™ :- Trademark symbol (™)
♥ :- Heart symbol (♥)
